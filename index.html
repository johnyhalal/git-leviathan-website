Features

Clone from anywhere

Connect your GitHub and GitLab accounts, or paste any Git URL to clone a repository in seconds.

Native on every desktop

A fast, focused window that runs natively on macOS, Windows, and Linux — no browser tab, no distractions, just your repositories.

Visual commit graph

A topo-ordered graph with branch and tag leader lines, your uncommitted working tree, and stashes all woven in — click any commit for its files, metadata, and author avatars.

Stage, commit & rewrite

Review side-by-side diffs, stage or discard changes, then commit, amend, reword, or push in one step — with undo and redo for ref-changing operations.

Branch, merge & sync

Create, checkout, and delete local and remote branches, merge or rebase, run Gitflow starts and finishes, and push or pull with a configurable mode.

Always in sync

GitLeviathan watches your working tree and re-syncs the UI on external edits, commits, and window focus — with live git activity streamed to a footer log.

Secure by default

Generate ed25519 SSH keys, connect accounts over OAuth device flow, and clone with authenticated URLs — secrets are always redacted from anything shown on screen.

AI-drafted commits

Turn your staged diff into a clear commit message with a single click, powered by your locally installed claude CLI — no credentials stored in the app.

Light & dark themes

A clean interface that follows your system appearance, or switch between light and dark whenever you like.
